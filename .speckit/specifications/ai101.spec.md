  Source: https://learn.microsoft.com/azure/ai-services/agents/overview

- **Agents are built to operate inside enterprise systems.** Agent platforms manage identity, authentication, and secure tool access (for example, API calls and data sources), reinforcing why governance and guardrails are essential as autonomy increases.  
  Source: https://learn.microsoft.com/azure/ai-services/agents/overview

- **Copilot agents are grounded in organizational data and permissions.** When agents work with Microsoft 365 data, they only surface information users are authorized to access, highlighting permissioning as a core control mechanism.  
  Source: https://learn.microsoft.com/microsoft-365-copilot/microsoft-365-copilot-overview

- **Observability becomes critical as autonomy grows.** Agent services provide end-to-end tracing and monitoring so teams can inspect decisions and tool calls — a capability far less critical for simple chat, but essential for agents.  
  Source: https://learn.microsoft.com/azure/ai-services/agents/overview
